# 挖金矿 (Gold Miner)

AI受益股"金矿股"挖掘与评估系统。

## 简介

从650+只AI受益股中，通过系统化的7维度评估体系，挖掘出真正的"金矿股"——即AI浪潮中受益确定性高、成长空间大、估值合理的优质标的。

## 核心特性

- **7维度评估体系**：AI含量、护城河、需求增长、财务质量、估值水平、市值/AI利润比、催化剂/风险
- **双轨评级制度**：AI自动评分 + 用户最终评级
- **每日考察模式**：每天推送10只待考察股票
- **数据驱动决策**：结合脚本自动获取 + LLM智能分析

## 评估维度

| 维度 | 权重 | 说明 |
|-----|------|------|
| AI含量 | 25% | AI业务收入占比、客户质量 |
| 护城河 | 20% | 市占率、技术壁垒、客户粘性 |
| 需求增长 | 20% | 行业增速、订单增长 |
| 财务质量 | 15% | ROE、毛利率、现金流 |
| 市值/AI利润比 | 10% | AI业务专项估值 |
| 估值水平 | 5% | PE、PEG、历史分位 |
| 催化剂/风险 | 5% | 近期事件、潜在风险 |

## 评级标准

| 评级 | 总分区间 | 含义 |
|-----|---------|------|
| ⭐⭐⭐⭐⭐ | 4.5-5.0 | 金矿股：强烈推荐 |
| ⭐⭐⭐⭐ | 4.0-4.4 | 优质股：推荐考察 |
| ⭐⭐⭐ | 3.0-3.9 | 中等股：可选考察 |
| ⭐⭐ | 2.0-2.9 | 一般股：暂不推荐 |
| ⭐ | <2.0 | 低质量：不推荐 |
