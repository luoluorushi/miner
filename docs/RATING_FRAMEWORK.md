# AI受益股评级框架 (Gold Mine Rating Framework)

## 一、核心理念

### 目标
从AI受益股池中系统化挖掘"金矿股"——即AI浪潮中受益确定性高、成长空间大、估值合理的优质标的。

### 工作流程
```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  核心股票池  │ ──▶ │  维度信息采集 │ ──▶ │   AI评分    │ ──▶ │  用户评级   │
│  (650+只)   │     │  (脚本+LLM)  │     │  (规则+LLM) │     │  (最终决策) │
└─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘
```

---

## 二、评估维度体系 (7大维度)

### 维度1：AI含量 (AI Exposure) - 权重 25%

**定义**：公司业务与AI的关联程度，AI收入占比

| 评分 | 标准 | 示例 |
|-----|------|------|
| 5分 | AI业务占收入>50%，或为AI核心基础设施 | 英伟达、寒武纪、中际旭创 |
| 4分 | AI业务占收入20-50%，明确受益于AI订单 | 沪电股份、工业富联 |
| 3分 | AI业务占收入10-20%，有明确AI产品线 | 立讯精密、鹏鼎控股 |
| 2分 | AI业务占收入<10%，但有布局或间接受益 | 传统PCB企业转型中 |
| 1分 | AI业务占比极小，概念炒作为主 | 纯蹭概念股 |

**信息获取方式**：
- 财报中AI相关业务收入披露
- 投资者关系活动记录
- 券商研报中的AI收入拆分
- 客户结构分析（是否供货给英伟达、微软、谷歌等）

---

### 维度2：护城河 (Moat) - 权重 20%

**定义**：公司在其领域的竞争壁垒深度

| 评分 | 标准 | 护城河类型 |
|-----|------|-----------|
| 5分 | 全球垄断或寡头，技术代差明显 | ASML(光刻机)、台积电(先进制程) |
| 4分 | 国内龙头+全球前三，有技术壁垒 | 中际旭创(800G光模块)、北方华创 |
| 3分 | 行业前五，有一定规模或技术优势 | 沪电股份、胜宏科技 |
| 2分 | 有竞争力但壁垒不高，容易被替代 | 中小型PCB厂商 |
| 1分 | 完全竞争市场，无明显护城河 | 同质化严重的代工厂 |

**护城河类型判断**：
- **技术壁垒**：专利数量、研发投入占比、良率优势
- **规模效应**：市占率、产能规模、成本优势
- **客户粘性**：认证周期、切换成本、长期合作关系
- **品牌/渠道**：消费端品牌溢价、B端口碑

**信息获取方式**：
- 行业研报中的市占率数据
- 公司年报中的研发投入、专利情况
- 主要客户及合作年限
- 行业竞争格局分析

---

### 维度3：需求增长 (Demand Growth) - 权重 20%

**定义**：所在赛道的市场增速和公司的增长动能

| 评分 | 标准 | 市场增速(CAGR) |
|-----|------|---------------|
| 5分 | 爆发式增长，供不应求 | >50% (如800G光模块、AI服务器) |
| 4分 | 高速增长，景气上行 | 30-50% (如AI PCB、HBM) |
| 3分 | 稳健增长，行业上行周期 | 15-30% |
| 2分 | 缓慢增长或周期底部 | 5-15% |
| 1分 | 行业萎缩或严重过剩 | <5%或负增长 |

**信息获取方式**：
- 行业协会数据（如SEMI、Prismark）
- 公司订单/营收增速
- 产能利用率、扩产计划
- 下游客户资本开支计划

---

### 维度4：财务质量 (Financial Quality) - 权重 15%

**定义**：盈利能力、现金流、资产负债表健康度

| 评分 | 指标参考 |
|-----|---------|
| 5分 | ROE>20%, 毛利率>40%, 净利率>15%, 经营现金流充沛 |
| 4分 | ROE 15-20%, 毛利率30-40%, 净利率10-15% |
| 3分 | ROE 10-15%, 毛利率20-30%, 净利率5-10% |
| 2分 | ROE 5-10%, 盈利但利润率较薄 |
| 1分 | ROE<5%或亏损，财务状况堪忧 |

**关键指标**：
- ROE（净资产收益率）
- 毛利率、净利率
- 经营性现金流/净利润
- 资产负债率
- 应收账款周转

**信息获取方式**：
- 财务报表（脚本自动获取）
- 历史财务数据趋势

---

### 维度5：估值水平 (Valuation) - 权重 5%

**定义**：当前股价相对于基本面的合理性

| 评分 | 标准 |
|-----|------|
| 5分 | 明显低估，PE/PEG显著低于历史和行业 |
| 4分 | 相对低估，有安全边际 |
| 3分 | 估值合理，与增速匹配 |
| 2分 | 估值偏高，透支部分预期 |
| 1分 | 严重高估，泡沫明显 |

**评估方法**：
- PE vs 行业平均
- PEG（PE/增速）
- PS、PB（适用于不同行业）
- 历史估值分位

**信息获取方式**：
- 实时行情数据
- 历史估值区间
- 同行业估值比较

---

### 维度6：市值/AI利润比 (Market Cap to AI Profit) - 权重 10%

**定义**：公司市值与其AI业务利润的比值，衡量市场对AI业务的定价是否合理

**核心公式**：
```
市值/AI利润比 = 市值 ÷ (总利润 × AI收入占比)
             = 市值 ÷ AI相关利润
```

**为什么重要**：
- 传统PE无法区分AI业务与非AI业务的定价
- 有些公司市值高但AI业务占比小，实际AI业务被高估
- 能快速识别"伪AI概念股"与"真AI龙头"

| 评分 | 市值/AI利润比 | 含义 |
|-----|--------------|------|
| 5分 | <30x | AI业务明显低估，市场未充分认识 |
| 4分 | 30-50x | AI业务合理定价，有安全边际 |
| 3分 | 50-80x | AI业务估值合理，增速匹配 |
| 2分 | 80-150x | AI业务估值偏高，需高增速支撑 |
| 1分 | >150x | AI业务严重高估，或AI占比过低导致虚高 |

**计算示例**：

| 公司 | 市值 | 总利润 | AI占比 | AI利润 | 市值/AI利润 | 评分 |
|-----|------|-------|--------|--------|------------|------|
| 中际旭创 | 6000亿 | 71亿 | 85% | 60亿 | 100x | 2分 |
| 某传统公司 | 500亿 | 10亿 | 5% | 0.5亿 | 1000x | 1分 |
| 某低估龙头 | 300亿 | 20亿 | 50% | 10亿 | 30x | 5分 |

**评分逻辑**：
- 如果市值/AI利润比很高，说明市场给AI业务定价过高，或AI业务实际占比太低
- 如果市值/AI利润比很低，说明AI业务被低估，存在上涨空间
- 结合行业特性调整（如半导体设备行业可接受更高倍数）

**信息获取方式**：
- 市值：脚本自动获取
- 总利润：财务数据（脚本自动）
- AI收入占比：研报/财报分析（LLM搜索）
- 计算公式自动化

**注意事项**：
- AI收入占比需要从研报或公司披露获取，可能有估算误差
- 对于纯AI公司（AI占比>80%），此指标接近于PE
- 对于AI占比<10%的公司，此指标会非常高，需综合AI含量维度判断

---

### 维度7：催化剂/风险 (Catalyst & Risk) - 权重 5%

**定义**：短期内可能推动或阻碍股价的因素

| 评分 | 标准 |
|-----|------|
| 5分 | 重大利好催化剂明确，风险可控 |
| 4分 | 有潜在催化剂，风险较小 |
| 3分 | 催化剂和风险平衡 |
| 2分 | 催化剂不明确，存在一定风险 |
| 1分 | 重大风险悬而未决 |

**催化剂类型**：
- 新产品/新客户突破
- 产能释放
- 行业政策利好
- 业绩拐点

**风险类型**：
- 大客户依赖
- 技术迭代风险
- 地缘政治风险
- 解禁/减持压力
- 行业产能过剩

**信息获取方式**：
- 新闻公告
- 研报观点
- 股东变动
- 行业动态

## 三、评分计算

### 权重分配（7大维度）

| 维度 | 权重 | 说明 |
|-----|------|------|
| AI含量 | 25% | 核心指标，决定AI受益程度 |
| 护城河 | 20% | 竞争优势持久性 |
| 需求增长 | 20% | 成长空间和确定性 |
| 财务质量 | 15% | 基本面健康度 |
| 市值/AI利润比 | 10% | AI业务专项估值 |
| 估值水平 | 5% | 传统估值指标 |
| 催化剂/风险 | 5% | 短期驱动因素 |

### 总分公式
```
AI评分 = AI含量×0.25 + 护城河×0.20 + 需求增长×0.20 + 财务质量×0.15 
       + 市值AI利润比×0.10 + 估值水平×0.05 + 催化剂×0.05
```

### 评级划分

| 评级 | 总分区间 | 含义 |
|-----|---------|------|
| ⭐⭐⭐⭐⭐ | 4.5-5.0 | 金矿股：强烈推荐，优先考察 |
| ⭐⭐⭐⭐ | 4.0-4.4 | 优质股：推荐考察 |
| ⭐⭐⭐ | 3.0-3.9 | 中等股：可选考察 |
| ⭐⭐ | 2.0-2.9 | 一般股：暂不推荐 |
| ⭐ | <2.0 | 低质量：不推荐 |

## 五、脚本命令

### 数据库操作 (db.py)

```bash
# 查看排行榜
python db.py list --top 20 --sort ai

# 按板块筛选
python db.py list --sector 光通信

# 查看单只股票详情
python db.py get sz300308

# 添加新股票
python db.py add sz300308 中际旭创 光通信

# 更新维度评分
python db.py update sz300308 --dimension ai_exposure --score 5 --evidence "AI收入占比85%"

# 用户评级
python db.py rate sz300308 --stars 4 --notes "估值偏高"

# 查看待评级股票
python db.py pending --top 10

# 导出数据
python db.py export --format csv

# 查看统计
python db.py stats
```

### 股票数据获取 (stock-data)

```bash
# 搜索股票
stock-data search 中际旭创

# 公司简况
stock-data profile sz300308

# 财务数据
stock-data finance sz300308 summary

# 实时行情
stock-data quote sz300308

# 研报评级
stock-data rating sz300308

# 最新新闻
stock-data news sz300308
```
